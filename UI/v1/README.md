# UI v1

这是根据最新信息架构调整后的手机端前端界面原型。

## 文件

- `index.html`：移动端单手机界面
- `styles.css`：移动端样式
- `app.js`：页面跳转与主导航逻辑

## 当前结构

- 首页：朋友双列列表、顶部搜索框、右下角添加朋友按钮
- 朋友详情：头像、基本信息、自定义信息、最近记录
- AI 查询：基于当前朋友档案进行自然语言提问
- 我的：账号、设置、隐私、AI 配置
- 添加朋友：建立基础档案并补充自定义信息

## 当前交互

- 首页和我的通过底部导航切换
- 首页点击朋友卡片进入详情
- 详情点击“问问我”进入 AI 查询
- 首页右下角加号进入添加朋友

## 预览
